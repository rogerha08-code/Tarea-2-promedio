# Cálculo de Promedio de Calificaciones

## Descripción
Este proyecto consiste en una aplicación de escritorio que permite calcular el promedio de calificaciones de un estudiante.

La interfaz del programa cuenta con cuatro campos de texto (**TextBox**) donde el usuario introduce las calificaciones. El sistema suma las cuatro notas ingresadas, calcula el promedio dividiendo el total entre cuatro y muestra el resultado en pantalla.

El promedio calculado se presenta en un **Label** con el siguiente formato:  
**“El promedio es [promedio]”.**

---

## Funcionalidades
- Ingreso de cuatro calificaciones.
- Cálculo automático del promedio.
- Visualización del resultado en la interfaz.
- Interfaz sencilla y fácil de utilizar.

---

## Tecnologías Utilizadas
- **Lenguaje:** C#
- **Framework:** .NET (Windows Forms)
- **Entorno de desarrollo:** Visual Studio 2022

---

## Uso o Ejecución
Para ejecutar el proyecto se deben seguir los siguientes pasos:

1. Clonar el repositorio desde GitHub.
2. Abrir el archivo `.sln` en **Visual Studio 2022**.
3. Compilar el proyecto.
4. Ejecutar la aplicación presionando **F5** o el botón **Iniciar**.
5. Introducir las cuatro calificaciones y presionar el botón correspondiente para calcular el promedio.
